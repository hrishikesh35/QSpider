Block Level Elements
<div> </div>  ---> block Element
eg.
    <div style="border: 1px solid black; width: 100px; height: 40px;">Hrishikesh</div>
                         |    |     |             |              |                       
                 thickness  style  color         size           size 

<span> </span>  ---> inline Element
eg.
    <span style="border: 1px solid rgb(203, 12, 12);">Hrishikesh</span>

    Inline Element does ot take height and width, they accquire height and width.

NOTE: blocl Level Element are arrange vertically
      inline Element are arrange horizontaly

HW: 5 inline and block Elements

Here are 5 block elements and 5 inline elements in HTML:

Block elements:

<p> - defines a paragraph 
<div> - defines a division or a section 
<h1> - defines HTML headings 
<ol> - defines an ordered list 
<ul> - defines an unordered list 
Inline elements:

<a> - defines a hyperlink 
<span> - defines a span of text
<img> - defines an image 
<em> - defines emphasized text 
<strong> - defines important text
